TechEdu Africa

Techédu Africa est une platforme fictive d'une école pour promouvoir l'apprentissage de l'informatique.
Son objectif principale est de fournir une page interactive ou les élève peuvent découvrire et se souscrire à cete école.
Contexte:Ce projet a été ralisé pour pratiquer mes compétences en HTML/CSS.

Technologie utlisée:
  • Languages:HTML/CSS,
  • Outils: vscode, CSS gradiant générator
Mon rôle
  1. Crée la stucture HTML et les styles CSS pour le disign de la platforme.
  2. Concevoir une mise en page fluide et bien stucturé.
  3. Intégrer des images et des liens pour une meilleure compréhension.
Défis
  • le manqu de temp.
  • Adapter le design
  • Optimisé le code pouur une bonne expérience.

Réseaux

Ce projet vise a explorer les base de la configuration réseaux a l'aide de cisco pocket traceur,
un outils indispensable pour la compréhension.L'objectif principale étant de comprendre les notion de base comme le routage, de la communication et de la connectivité.
Contexte:ce projet a été réalisé dans un curs de réseaux informatique.

Technologies utilisées
  • Outils:cisco pocket tracer
  • Language:aucun
    • Mon rôle
    1. Crée un réseaux virtuelle avec des routeur.
    2. Configuré les paramètre IP.
    3. Testé la connectivité avec la commande Ping.
    Défis
    • Comprendre les conceptes de routage.
    • Configuré les commande IP.
    • Assuré une connectivitée.
